1.A Rn 和 Cn

本节概览

本节是第 1 章的起点，也是整本书的基石。从复数的定义出发，逐步构建出==Fn==（Rn 或 Cn）这一核心对象，并定义其上的加法和标量乘法两种运算。整个逻辑链条为：

复数定义 → 复数算术性质 → 减法与除法 → 记号 F → 组的定义 → Fn 的定义 → Fn 中的加法 → Fn 中的标量乘法

一、复数

1.1 复数的定义

定义 1.1：复数（complex numbers）、 C

一个复数是一个有序对 (a,b)，其中 a,b∈R，写成 a+bi 的形式。

全体复数所构成的集合用 C 表示： C={a+bi:a,b∈R}

C 上的加法和乘法定义为： (a+bi)+(c+di)=(a+c)+(b+d)i (a+bi)(c+di)=(ac−bd)+(ad+bc)i 其中 a,b,c,d∈R。

学习注解

  • 如果 a∈R，将 a+0i 等同于实数 a，从而 R⊂C。
  • 通常将 0+bi 简写作 bi，将 0+1i 简写作 i。
  • 复数乘法定义的来由：假设 i2=−1，用一般算术规则展开 (a+bi)(c+di)=ac+adi+bci+bdi2=(ac−bd)+(ad+bc)i，再用定义式验证 i2=−1 确实成立。
  • 不要背乘法公式——只需记住 i2=−1，再运用分配律和交换律即可重新推导。

1.2 复数的算术性质

1.3：复数的算术性质

可交换性（commutativity）：对所有 α,β∈C，有 α+β=β+α 以及 αβ=βα。

可结合性（associativity）：对所有 α,β,λ∈C，有 (α+β)+λ=α+(β+λ) 以及 (αβ)λ=α(βλ)。

恒等元（identities）：对所有 λ∈C，有 λ+0=λ 以及 λ⋅1=λ。

加法逆元（additive inverse）：对每个 α∈C，都存在唯一的 β∈C 使得 α+β=0。

乘法逆元（multiplicative inverse）：对每个 α∈C 且 α=0，都存在唯一的 β∈C 使得 αβ=1。

分配性质（distributive property）：对所有 λ,α,β∈C，有 λ(α+β)=λα+λβ。

学习注解

这六条性质可以用实数的性质加上复数加法、乘法的定义来证明。它们是复数作为”域”（field）的公理——后续所有关于复数的运算都建立在这六条性质之上。

1.3 例题

例 1.2：复数的算术运算

利用分配律和交换律计算 (2+3i)(4+5i)： (2+3i)(4+5i)=2⋅(4+5i)+(3i)(4+5i) =2⋅4+2⋅5i+3i⋅4+(3i)(5i) =8+10i+12i+15i2 =8+10i+12i−15 =−7+22i

例 1.4：复数乘法的可交换性

要说明对所有 α,β∈C 均有 αβ=βα，设 α=a+bi，β=c+di，其中 a,b,c,d∈R。

由复数乘法定义： αβ=(a+bi)(c+di)=(ac−bd)+(ad+bc)i βα=(c+di)(a+bi)=(ca−db)+(cb+da)i

结合实数乘法与加法的可交换性（ac=ca，bd=db，ad=da，bc=cb），可得 αβ=βα。

1.4 减法与除法

定义 1.5： −α、减法（subtraction）、1/α、除法（division）

假设 α,β∈C。

令 −α 表示 α 的加法逆元。于是 −α 是唯一使得 α+(−α)=0 成立的复数。

C 上的减法定义为：β−α=β+(−α)。

对于 α=0，令 1/α 和 α−1 表示 α 的乘法逆元。于是 1/α 是唯一使得 α(1/α)=1 成立的复数。

对于 α=0，除以 α 定义为：β/α=β(1/α)。

学习注解

减法和除法并非独立的运算——它们是通过加法逆元和乘法逆元从加法和乘法”派生”出来的。这种”定义基本运算 + 逆元 + 派生运算”的模式，在后续定义 Fn 中的减法时也会出现。

二、Fn 的定义

2.1 记号 F

记号 1.6： F

在全书中，F 代表 R 或 C。

学习注解

  • 如果我们证明了一个涉及 F 的定理，那么将 F 替换为 R 或 C 时定理也成立。
  • F 中的元素称为标量（scalar）。“标量”只是”数”的一个花哨表达法，用来强调它是数而非向量。
  • 字母 F 的使用是因为 R 和 C 都是域（field）的实例。
  • 对于 α∈F 和正整数 m，定义 αm=m 个 αα⋅⋯⋅α​​，由此可得 (αm)n=αmn 和 (αβ)m=αmβm。

2.2 组

定义 1.8：组（list）、长度（length）

假设 n 是非负整数。一个长度为 n 的组是 n 个有顺序的元素，这些元素可能是数、其他组或是更抽象的对象。

两个组相等，当且仅当它们具有相同的长度和按相同顺序排列的相同元素。

学习注解

  • 组的通常写法是将元素以逗号分隔并用圆括号括起来：(z1​,…,zn​)。
  • 长度为 2 的组是有序对 (a,b)，长度为 3 的组是有序三元组 (x,y,z)。
  • 每个组都具有有限长度（非负整数），因此 (x1​,x2​,…)（无限长度）不是组。
  • 长度为 0 的组是 ()，引入它是为了避免一些定理出现平凡的例外情形。

例 1.7： R2 和 R3

R2={(x,y):x,y∈R} 可视化为一个平面。

R3={(x,y,z):x,y,z∈R} 可视化为通常的三维空间。

例 1.9：组 VS 集合

组与有限集有两方面差异：在组中，顺序很重要，并且重复是有含义的；而在集合里，顺序和重复都无关紧要。

  • 组 (3,5) 和 (5,3) 不相等，但集合 {3,5} 和 {5,3} 相等。
  • 组 (4,4) 和 (4,4,4) 不相等（长度不同），但集合 {4,4} 和 {4,4,4} 都等于集合 {4}。

2.3 Fn 的定义

记号 1.10： n

在本章剩余内容中，将 n 取为某一固定的正整数。

定义 1.11： Fn、坐标（coordinate）

Fn 是全体具有 n 个 F 中元素的组所构成的集合： Fn={(x1​,…,xn​):对于 k=1,…,n 有 xk​∈F}

对于 (x1​,…,xn​)∈Fn 和 k∈{1,…,n}，称 xk​ 是 (x1​,…,xn​) 的第 k 个坐标。

例 1.12： C4

C4={(z1​,z2​,z3​,z4​):z1​,z2​,z3​,z4​∈C} C4 的每个元素是一个由 4 个复数组成的组。

学习注解

  • 如果 F=R 且 n=2 或 3，则 Fn 的定义与 R2、R3 的定义吻合。
  • 当 n≥4 时，无法将 Rn 可视化为物理实体；当 n≥2 时，人脑也不能想象出 Cn 的全貌。然而，即便 n 很大，我们也可以如在 R2 或 R3 中那样简便地在 Fn 中进行代数运算。因此，这门学科叫做线性代数。

三、Fn 中的加法

3.1 加法的定义

定义 1.13： Fn 中的加法（addition in Fn）

Fn 中的加法定义为将对应坐标分别相加： (x1​,…,xn​)+(y1​,…,yn​)=(x1​+y1​,…,xn​+yn​)

3.2 加法的可交换性

命题 1.14： Fn 中加法的可交换性

如果 x,y∈Fn，那么 x+y=y+x。

证明思路

设 x=(x1​,…,xn​)∈Fn 且 y=(y1​,…,yn​)∈Fn。

[关键步骤] 逐步展开坐标并利用 F 中加法的可交换性： x+y=(x1​,…,xn​)+(y1​,…,yn​) =(x1​+y1​,…,xn​+yn​)——由 Fn 中加法的定义 =(y1​+x1​,…,yn​+xn​)——由 F 中加法的可交换性 =(y1​,…,yn​)+(x1​,…,xn​)——由 Fn 中加法的定义 =y+x

■

学习注解

证明的核心模式：将 Fn 中的性质归结为 F 中的性质。Fn 中加法的可交换性并非独立的新性质，而是直接从 F（R 或 C）中加法的可交换性”继承”而来的。这种”逐坐标验证”的证明模式将在整本书中反复出现。

3.3 零向量

记号 1.15： 0

令 0 表示长度为 n 且所有坐标都是 0 的组： 0=(0,…,0)

例 1.16：根据上下文确定使用的是哪种 0

考虑”0 是 Fn 中的加法恒等元”这一表达式： 对所有 x∈Fn，有 x+0=x

此处式中的 0 是==组 0∈Fn==，而不是数 0，因为我们并未定义 Fn 中元素（此处即 x）和数 0 的加法。

学习注解

符号 0 的多重含义是初学者常见的困惑来源。在本书中，0 至少可以表示：

  • 数 0∈R（或 C）
  • 零向量 0∈Fn
  • 零映射 0∈L(V,W)（后续章节）

根据上下文区分——看 0 参与的是什么运算、和什么类型的对象在一起。

3.4 加法逆元

定义 1.17： Fn 中的加法逆元（additive inverse in Fn）、−x

对于 x∈Fn，x 的加法逆元，记作 −x，是满足下式的向量 −x∈Fn： x+(−x)=0

如果 x=(x1​,…,xn​)，那么 −x=(−x1​,…,−xn​)。

学习注解

在 R2 中，向量 −x 与 x 长度相同但指向相反方向。减法可以类似定义为 y−x=y+(−x)。

四、Fn 中的标量乘法

4.1 标量乘法的定义

定义 1.18： Fn 中的标量乘法（scalar multiplication in Fn）

数 λ 与 Fn 中的向量之乘积是通过将这向量的每一个坐标都乘以 λ 计算得到的： λ(x1​,…,xn​)=(λx1​,…,λxn​) 其中 λ∈F 且 (x1​,…,xn​)∈Fn。

学习注解

  • 标量乘法将一个标量（F 中的数）和一个向量（Fn 中的元素）相乘，得到一个向量。
  • 这与点积（dot product）不同：点积将两个向量相乘，得到一个标量。点积的推广形式将在第 6 章中变得重要。
  • Axler 明确指出：本可以定义 Fn 中两个元素的逐坐标乘法，但”经验表明，这种定义无助于实现我们的目的”。标量乘法才是线性代数的核心运算。

4.2 几何解释

几何直觉（ R2 中的可视化）

向量的两种视角：

  • 点：v=(a,b) 是平面上的一个点
  • 箭头：从原点出发、到 (a,b) 结束的箭头（此时称为向量）

箭头可以平行移动（不改变长度和方向），仍视为同一个向量。

加法的几何意义：将向量 v 平行移动使其起点与 u 的终点重合，则 u+v 是从 u 的起点到 v 的终点的向量。

标量乘法的几何意义：

  • 若 λ>0：λx 与 x 同方向，长度为 x 的 λ 倍（缩短或延长）
  • 若 λ<0：λx 与 x 反方向，长度为 x 的 ∣λ∣ 倍
  • 若 λ=0：λx=0（零向量）

六、核心思想与证明技巧

核心思想

  1. 复数是有序对：C 不是”虚幻的数”，而是有序对 (a,b) 配上特定的加法和乘法规则。i2=−1 是定义的推论，而非假设。
  2. Fn 是组的集合：Fn 的元素是长度为 n 的组（有序、可重复），不是集合。这是线性代数中”顺序”重要性的第一个体现。
  3. 逐坐标定义运算：Fn 中的加法和标量乘法都是逐坐标定义的，这使得 Fn 中的代数性质直接继承自 F 中的性质。
  4. 代数优于几何：虽然低维（n=2,3）的几何直觉有助于理解，但线性代数的真正力量在于代数方法对任意 n 都有效。
  5. 标量乘法是核心：逐元素乘法被放弃，标量乘法（数乘向量）成为基本运算——这是线性结构（“缩放”）的本质。

证明技巧清单

  • 逐坐标验证法：将 Fn 中的性质归结为 F 中的性质（如命题 1.14 的证明）
  • 利用 i2=−1 重新推导：不要背复数乘法公式，用分配律 + i2=−1 现场推导
  • 上下文区分符号：遇到 0 时，先判断它参与的是什么运算、和什么类型的对象在一起

七、补充理解与易混淆点

为什么需要复数？

复数的发明是为了解决”负数不能开平方”的问题。更深刻地说，复数使得代数基本定理成立：每个非常数复系数多项式都有复数根。这意味着在 C 上，所有多项式都可以完全分解为一次因式的乘积——这是 R 做不到的。

来源：Purdue University 讲义。

Fn 的几何直觉——从 R2 到高维

R2 可以可视化为平面，R3 可以可视化为三维空间。但当 n≥4 时，==我们无法将 Rn 可视化为物理实体==。然而，即便 n 很大（如经济学中的 R5000），我们仍然可以在 Fn 中进行代数运算——这正是”线性代数”的真正含义：用代数方法处理高维空间。

来源：Axler 原文、UW-Madison 讲义。

组与集合的本质区别

组（list）集合（set）
顺序重要：(3,5)=(5,3)无关：{3,5}={5,3}
重复有含义：(4,4)=(4)无关：{4,4}={4}
长度有限非负整数可无限
用途坐标、向量表示元素归属

域的概念——为什么是 R 和 C？

域是具有加法和乘法运算、满足 1.3 中所有性质的集合。R 和 C 是最常见的域，但还有其他域：

  • Q（有理数域）
  • F2​={0,1}，其中 1+1=0（有限域，可能导致直觉失效）

在本书中，F 代表 R 或 C，但多数定理对任意域都成立。

来源：Axler 原文”域的题外话”、Northeastern Dummit 讲义。

向量的两种视角——点 vs 箭头

R2 中的元素可以看作：

  • 点：(a,b) 是平面上的一个位置
  • 向量（箭头）：从原点出发、到 (a,b) 结束的有向线段

箭头可以平行移动（不改变长度和方向），仍视为同一向量。加法对应”首尾相接”，标量乘法对应”缩放”。

符号 0 的多重含义

在线性代数中，符号 0 可能表示：

  • 标量 0∈F（数零）
  • 零向量 0=(0,…,0)∈Fn
  • 零空间 {0}
  • 零映射

具体含义由上下文决定。例如 "x+0=x" 中，如果 x∈Fn，则 0 是零向量；如果 x∈F，则 0 是数零。

注意

这些几何直觉仅适用于 R2 和 R3。对于 Cn 或 n≥4，几何方法失效，必须依赖代数。

常见误区

误区1："向量就是有方向的箭头"

❌ 函数、多项式、矩阵、无穷序列都可以是”向量”。向量的本质不是几何表示，而是属于某个集合且支持加法和数乘运算。

来源：Northeastern Dummit 讲义。

误区2：" i 是虚数所以不存在"

❌ 复数 a+bi 不是”虚幻的数”，而是有序对 (a,b) 配上特定的加法和乘法规则。i2=−1 是定义的推论，而非假设。复数在物理学、工程学、信号处理中有广泛的真实应用。

来源：Purdue University 讲义。

误区3：" R2 是 R3 的子空间"

❌ R2 的元素是二元组，R3 的元素是三元组——它们根本不是同一个集合的元素。不过 {(x,y,0):x,y∈R} 是 R3 的子空间，且与 R2 同构。

来源：University of Utah 讲义。

误区4："组就是集合"

❌ 组中顺序重要且重复有含义：(3,5)=(5,3)，(4,4)=(4)。集合中顺序无关且重复无含义：{3,5}={5,3}，{4,4}={4}。

来源：Axler 例 1.9。

误区5："标量乘法就是逐元素相乘"

❌ 标量乘法是将一个标量（数）乘以一个向量，得到一个向量：λ(x1​,…,xn​)=(λx1​,…,λxn​)。这与”两个向量逐元素相乘”（Hadamard 乘积）完全不同。标量乘法是线性结构的本质运算。

因此 i 的两个平方根为 2​1​+2​1​i 和 −2​1​−2​1​i。■

习题 9：求解 R4 中的向量方程

求 x∈R4 使得 (4,−3,1,7)+2x=(5,9,−6,8)。

查看解答

由 2x=(5,9,−6,8)−(4,−3,1,7)=(1,12,−7,1)，得 x=(21​,6,−27​,21​) ■

习题 10：标量乘法的无解情形

解释为什么不存在 λ∈C 使得 λ(2−3i, 5+4i, −6+7i)=(12−5i, 7+22i, −32−9i)

查看解答

由标量乘法的定义，λ(2−3i)=12−5i，由此解出： λ=2−3i12−5i​=(2−3i)(2+3i)(12−5i)(2+3i)​=4+924+36i−10i−15i2​=1324+26i+15​=1339+26i​=3+2i

验证第二个分量：(3+2i)(5+4i)=15+12i+10i+8i2=15+22i−8=7+22i ✓

验证第三个分量：(3+2i)(−6+7i)=−18+21i−12i+14i2=−18+9i−14=−32+9i

但题目要求第三个分量为 −32−9i，而实际得到 −32+9i=−32−9i。

矛盾：不存在 λ 能同时满足三个分量。■

习题 11： Fn 中加法的可结合性

证明 (x+y)+z=x+(y+z) 对所有 x,y,z∈Fn 成立。

查看解答

设 x=(x1​,…,xn​)，y=(y1​,…,yn​)，z=(z1​,…,zn​)。

(x+y)+z=((x1​+y1​,…,xn​+yn​))+(z1​,…,zn​) =(x1​+y1​+z1​,…,xn​+yn​+zn​)

x+(y+z)=(x1​,…,xn​)+((y1​+z1​,…,yn​+zn​)) =(x1​+y1​+z1​,…,xn​+yn​+zn​)

两者相等，其中用到了 F 中加法的可结合性。■

习题 15：标量乘法对加法的分配性

证明 (a+b)x=ax+bx 对所有 a,b∈F 和 x∈Fn 成立。

查看解答

设 x=(x1​,…,xn​)。

(a+b)x=(a+b)(x1​,…,xn​)=((a+b)x1​,…,(a+b)xn​)

ax+bx=(ax1​,…,axn​)+(bx1​,…,bxn​)=(ax1​+bx1​,…,axn​+bxn​)

由 F 中乘法对加法的分配性（(a+b)xk​=axk​+bxk​），两者相等。■
